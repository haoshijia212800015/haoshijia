# 好食家 - 智能饮食助手

一个面向个人用户的极简智能饮食推荐工具，帮助用户解决"日常该吃什么"的决策痛点。

## 🌐 在线访问
访问地址：https://美食家.github.io/haoshijia/

## 📱 功能特点

### 5个完整页面
1. **首页** - 品牌展示 + 核心功能入口
2. **信息填写页** - 个性化信息收集表单
3. **结果展示页** - 个性化饮食建议生成
4. **随机吃什么页** - 随机菜品推荐
5. **我的页** - 用户个人中心

### 核心功能
- 🎯 **智能推荐**：基于性别、年龄、目标、偏好生成个性化饮食方案
- 🎲 **随机推荐**：从8种菜品数据库中随机推荐，支持收藏
- 💾 **方案保存**：模拟保存功能
- ❤️ **收藏功能**：收藏喜欢的菜品
- 📤 **分享功能**：一键分享应用给朋友

### 技术特点
- 单文件HTML应用，无外部依赖
- 响应式设计，完美适配手机端
- PWA支持，可添加到主屏幕
- 纯原生JavaScript/CSS实现
- 离线可用

## 🚀 快速开始

### 直接使用
1. 用手机浏览器访问：https://美食家.github.io/haoshijia/
2. 点击浏览器菜单 → "添加到主屏幕"
3. 像原生应用一样使用

### 本地运行
1. 下载 `index.html` 文件
2. 双击在浏览器中打开
3. 所有功能本地运行，无需网络

## 📖 使用说明

### 智能推荐流程
1. 首页 → 点击"智能推荐"
2. 填写个人信息（性别、年龄、目标、偏好）
3. 点击"开始分析"
4. 查看个性化饮食建议

### 随机推荐
1. 首页 → 点击"随机吃什么"
2. 点击"随机推荐一道菜"
3. 可以"换一个"或"收藏"喜欢的菜品
