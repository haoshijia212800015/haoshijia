# 🚀 好食家生鲜PWA - 立即部署指南

## ⏱️ 5分钟快速部署

### 最简单的方法：拖拽到Vercel

1. **访问** [vercel.com](https://vercel.com)
2. **登录/注册**（可用GitHub账号）
3. **点击** "New Project"
4. **拖拽** 本文件夹到上传区域
5. **点击** "Deploy"

### 部署后操作

1. **复制** 生成的URL（如 `https://haoshijia-pwa.vercel.app`）
2. **手机访问** 该URL
3. **安装应用**：
   - iOS：Safari → 分享 → 添加到主屏幕
   - Android：Chrome → 菜单 → 添加到主屏幕

## 🆘 遇到问题？

### 常见问题：
1. **无法部署**：检查网络连接，重新尝试
2. **无法安装**：确保使用HTTPS，与网站有足够交互
3. **功能异常**：清除浏览器缓存，重新加载
