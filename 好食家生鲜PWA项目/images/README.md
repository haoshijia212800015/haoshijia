## 图片要求

### 图标文件
- 格式：PNG（透明背景）
- 设计：使用好食家Logo，主色为绿色(#4CAF50)
- 尺寸：必须包含所有指定尺寸
- 圆角：建议使用圆角设计，增强现代感

### 轮播图
- 格式：JPG或PNG
- 尺寸：建议750x300像素（2.5:1比例）
- 内容：展示平台核心业务和优势
- 风格：明亮、清新、有食欲感

### 商品图片
- 格式：JPG或PNG
- 尺寸：建议400x400像素（1:1比例）
- 内容：商品实拍图，背景干净
- 要求：图片清晰，色彩真实

## 替换说明

在实际部署前，请替换此目录下的所有占位图片：

1. **生成图标**：使用设计工具生成所有尺寸的图标
2. **制作轮播图**：设计3张轮播图，对应三大业务
3. **拍摄商品图**：为每个商品拍摄高质量图片
4. **制作截图**：在真实设备上截图，用于PWA展示

## 图片优化建议

1. **压缩图片**：使用工具压缩图片，减少文件大小
2. **WebP格式**：考虑使用WebP格式以获得更好的压缩效果
3. **懒加载**：页面已实现图片懒加载
4. **响应式图片**：可根据需要提供不同尺寸的图片
