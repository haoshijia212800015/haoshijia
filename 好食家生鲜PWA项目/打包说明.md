## 打包步骤

### 方法一：使用ZIP工具打包
1. 选中「好食家生鲜PWA项目」文件夹
2. 右键选择「发送到」→「压缩(zipped)文件夹」
3. 重命名为「好食家生鲜PWA项目.zip」

### 方法二：使用命令行打包
```bash
# 进入项目目录
cd "好食家生鲜PWA项目"

# 创建ZIP文件（Windows）
powershell Compress-Archive -Path * -DestinationPath ../好食家生鲜PWA项目.zip

# 创建ZIP文件（macOS/Linux）
zip -r ../好食家生鲜PWA项目.zip .
```

### 方法三：使用7-Zip等工具
1. 安装7-Zip或WinRAR
2. 右键项目文件夹 → 7-Zip → 添加到压缩包
3. 选择ZIP格式，压缩级别：标准

## 问题排查

### 常见问题1：Service Worker不工作
**症状**：无法离线访问，无法安装
**解决**：
1. 检查是否使用HTTPS（localhost除外）
2. 检查Service Worker文件路径
3. 清除浏览器缓存重新测试

### 常见问题2：无法添加到主屏幕
**症状**：浏览器不显示安装提示
**解决**：
1. 检查manifest.json格式
2. 确保用户与网站有足够交互
3. 使用Chrome的"添加到主屏幕"测试

### 常见问题3：页面样式错乱
**症状**：布局混乱，样式不生效
**解决**：
1. 检查CSS文件路径
2. 检查浏览器控制台错误
3. 清除缓存重新加载
